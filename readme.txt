== Description ==

[Facebook-Twitter-Wp-Login](https://jploft.com) plugin allows you to login your wordpress site via users facebook and twitter account.


=== Requirements ===

* A self-hosted website running on [WordPress hosting]

=== Facebook-Twitter-Wp-Login Settings ===

* **Select Application**: Your Selected Application (e.g. Facebook or Twitter).
* **Client ID(FB)/API key(TW)**:  Add your Client ID(FB)/API key(TW).
* **Client Secret(FB)/API secret key(TW):**: The Client Secret(FB)/API secret key(TW):.


=== Facebook-Twitter-Wp-Login Login ===

Once you have configured the settings you can allow user via their facebook and twitter account.
 


== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (Facebook-Twitter-Wp-Login) on your computer
1. Click "Install Now" and then hit the activate button
